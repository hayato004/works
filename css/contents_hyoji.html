コンテンツの表示方法の変更-おさまらないコンテンツにはスクロール-

要素の高さを指定するプロパティ・・・「 height: ～(単位を伴う数値が入る); 」
収まらないコンテンツの表示方法を指定するプロパティ・・・「overflow: ～(表示方法を表すキーワードが入る);」

縦に長くなってしまったサイドバーなどの項目の縦幅を縮めるためスクロールできるようにする。
縦方向のoverflowプロパティの値にscrollを指定すると指定した部分の高さを超える部分はスクロールで見れるようにできる。
例:
.recentNav ul {
height: 240px;
overflow: hidden scroll;
}

「あふれたコンテンツをどうするか？」を指定するoverflow
widthやheightで大きさを指定した要素からコンテンツがあふれてしまうときにどのように表示するかを指定するのがoverflowプロパティである。
overflowの書き方・・・「overflow: hidden(横方向の指定) scroll(縦方向の指定);」

横方向のみを指定 overflow-x: hidden;
縦方向のみを指定　overflow-y: scroll;

<主に使用する値一覧>
visible(※初期値)・・・はみ出して表示される。
hidden・・・非表示になる。
scroll・・・スクロールできるようになる。
auto・・・ブラウザに依存した表示になる。(※多くの場合、スクロールになる)
