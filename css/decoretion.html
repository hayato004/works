マウスカーソルをあてた時(マウスオーバー)の装飾

マウスカーソルをあてた時(マウスオーバー)の装飾を変える

■テキストの傍線を一括指定するプロパティ
text-decoration:～;
値(～の部分)には位置・線の種類・色指定・線の太さが入る。
カーソルをあてた時の指定は:hoverという疑似クラスを使う。

例：
header nav ul li a:hover {
text-decoration: underline double;
}

意味：
header内のnav内のul要素のliのみ、マウスオーバー(カーソルを合わせた時に)下二重線が出るようにする。

text-decorationのショートハンドの書き方

text-decorationはテキストに適用する傍線の位置・種類・色を一括で指定ができる。
■ショートハンドの書き方
text-decoration:underline(線の位置) double(線の種類) #ed7a92(線の色) 2px(線の太さ);
・値は半角スペースで区切る！！
・順番は自由！！

線の位置に指定できる値

underline(下線)
overline(上線)
line-through(取り消し線)
none(線無し)

線の種類に指定できる値

solid(実線)
double(二重線)
dotted(点線)
dashed(破線)
wavy(波線)

線の位置以外は省略可能。省略すると、種類はsolid(実線)、色は文字色と同じになり、太さはブラウザ依存になる。
