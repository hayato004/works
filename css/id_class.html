<id>属性ページ内に一つだけ存在する。(個別の識別番号の役割を果たすため。番号や名前を変えるならOK)

例
<p id="g018a1576">

文字の太さを変える場合、<id>タグで指定した後に、半角スペースのあとに、指定する。

例
<p id="g018a1576" class="bold">

このように、<id>タグを付与した場合、<style>の中の個別の識別番号の始まりは「#」から始める。

<class>タグはhtml内で同じ名前で複数つけられる。

例
<p class="blue_color">

文字の太さを変える場合、<color>タグで指定した後に、半角スペースの後に、bold big_fontを指定する。

例
<p class="blue_color bold big_font">

このように、<class>タグを付与した場合、<style>の中の個別の識別番号の始まりは「.」から始める。

colorは0が黒、fが白。#000000をRGBで分けて考え、最初の#00がR。次 の00に、最後の00にBをおいて考える。
