■font-styleについて■

<font-family(フォントファミリー)：~；>

カンマで区切って複数のフォントを指定することができる。

↓書き方↓

セレクタ{プロパティ名：値；}

サイトの文書全体に指定したいのでセレクタはbodyになる。

例：
body {

background-color:#bbf1ef;
font-family: "Verdana", "Hiragino Sans", "Meiryo", "sans-serif";
}

~font-family：の具体的な書き方について~
font-family:フォントファミリ名1(優先度1), フォントファミリ名2(優先度2), 総称ファミリ名(優先度3);
①フォントファミリ名は,（カンマ）で区切り、複数指定できる。
②フォントファミリ名と総称ファミリ名を正しく認識させるためにはフォントファミリ名を'（シングルクォーテーション）または、”（ダブルクォーテーション）で囲む必要がある。
③フォントは先に書いたものが優先される。フォントが無い場合、次に優先度2のフォント、それも無ければ次...という順番で適用される。最後に総称ファミリ名を書くと、最低限の見栄えを担保できる。

???総称ファミリ名とは???
字形で分類したフォント名。
指定されたフォントファミリがどれも存在しなかった場合の代替えフォントになる。

???指定するフォント名は何でも良いのか???
自分のPCでは問題なく表示されるフォント名を指定したとしても、サイトを見るユーザーのデバイス次第では総称に無いフォントは表示されない。総称ファミリ名
を指定する そのため、様々なデバイスでの見方を考慮する必要があり、一般的に複数のフォントファミリ名と総称ファミリ名を指定する。

■fontのサイズ変更について■

<font-size(フォントサイズ)：~；>

値にはpx(ピクセル)などの数値や、large(大きい)などの大きさを表すキーワードが使われます。

例：
h1 {

font-size: 18px;
}
